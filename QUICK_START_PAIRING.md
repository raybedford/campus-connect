# 📱 Quick Start: Device Pairing

## Transfer Keys in 3 Easy Steps!

---

## Mobile → Desktop (Most Common)

### On Desktop:
1. Settings → **QR Transfer** → **"Pair with Mobile"**
2. See QR code + 6-character code
3. Wait...

### On Mobile:
1. Open **Camera app**
2. Scan QR code
3. Verify code matches
4. Tap **"Transfer My Keys"**

### Done! ✅
Desktop automatically receives keys!

---

## Without Camera (Copy/Paste)

### Source Device:
1. Settings → QR Transfer → **"Export Keys"**
2. Click **"Copy to Clipboard"**
3. Send to yourself (Email/iMessage/AirDrop)

### Destination Device:
1. Settings → QR Transfer → **"Paste Keys"**
2. Paste the text (Cmd+V or Ctrl+V)
3. Click **"Import Keys"**

---

## ⏱️ Important
- **5 minute** time limit
- Verify 6-character code
- One-time use only

## 🔒 Security
- Keys encrypted during transfer
- QR codes expire after 15 minutes
- Code verification prevents wrong device
