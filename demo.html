Core Features
Built for how students
actually communicate

Every feature designed around the real needs of campus life — private by default, fast, and school-scoped.

🔒

End-to-End Encryption

NaCl Box encryption (Curve25519 + XSalsa20-Poly1305). Messages are encrypted on your device before they ever leave. The server stores only opaque blobs — it can never read your conversations.

⚡

Real-Time Messaging

Native WebSocket connections deliver messages instantly. Typing indicators, online presence, and read receipts keep conversations flowing naturally. Exponential backoff handles disconnects gracefully.

🎓

.edu Verified Identity

Sign up requires a valid .edu email. The system auto-detects your university and scopes your network. You can only discover and message students at your own school.

🔥

Ephemeral File Sharing

Files are encrypted client-side with a unique symmetric key, then wrapped per-recipient. After all recipients download, the file is automatically deleted from the server. 24-hour safety net cleanup.

👥

DMs & Group Chats

One-to-one DMs are automatically deduplicated. Group chats support named conversations with add/remove member controls. All messages encrypted per-recipient, even in groups.

📱

Mobile Ready

Responsive design works on any screen. Capacitor configuration ready to wrap as a native iOS or Android app — one codebase, every platform.

Security Model
The server sees
nothing

Messages are encrypted on the sender's device using the recipient's public key. The server only stores encrypted blobs, timestamps, and metadata. Private keys never leave your device — they're stored in IndexedDB, inaccessible to the server.

Algorithm: NaCl Box — Curve25519 key exchange + XSalsa20-Poly1305
Key Storage: IndexedDB via idb-keyval (never transmitted)
Library: TweetNaCl.js (audited, zero dependencies)
Sender nacl.box(plaintext, nonce, recipientPK, senderSK)
Ciphertext x8Kf2mP9...aQ7bR3nVw== (opaque to server)
Nonce Ym4xZjk3...NTJhYmI2
Stored {recipient_id, ciphertext_b64, nonce_b64}
Receiver nacl.box.open(ciphertext, nonce, senderPK, recipientSK)
